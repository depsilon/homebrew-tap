# Homebrew Tap For ShardLoom

This tap publishes the ShardLoom CLI formula.

```sh
brew install depsilon/tap/shardloom
```

ShardLoom v0.1.0 is a technical preview. Package access does not imply
production readiness, performance superiority, Spark replacement, broad
SQL/DataFrame support, object-store/lakehouse support, Foundry production
support, or fallback execution.
